electricpy.zpu¶

electricpy.zpu(S, VLL=None, VLN=None)¶

Per-Unit Impedance Evaluator

Evaluates the per-unit impedance value given the per-unit power and voltage bases.

\[Z_{pu}=\frac{V_{LL}^2}{S}\]
\[Z_{pu}=\frac{(\sqrt{3}*V_{LN})^2}{S}\]
Parameters

  • S (float) – The per-unit power base.

  • VLL (float, optional) – The Line-to-Line Voltage; default=None

  • VLN (float, optional) – The Line-to-Neutral Voltage; default=None

Returns

Zbase – The per-unit impedance base.

Return type

float
