electricpy.capenergy¶

electricpy.capenergy(C, V)¶

Capacitor Energy Formula

A simple function to calculate the stored voltage (in Joules) in a capacitor with a charged voltage.

\[E=\frac{1}{2}*C*V^2\]
Parameters

  • C (float) – Capacitance in Farads.

  • V (float) – Voltage across capacitor.

Returns

energy – Energy stored in capacitor (Joules).

Return type

float
