electricpy.captransfer¶

electricpy.captransfer(t, Vs, R, Cs, Cd)¶

Capacitor Energy Transfer Function

Calculate the voltage across a joining resistor (R) that connects Cs and Cd, the energy-source and -destination capacitors, respectively. Calculate the final voltage across both capacitors.

Parameters

  • t (float) – Time at which to calculate resistor voltage.

  • Vs (float) – Initial voltage across source-capacitor (Cs).

  • R (float) – Value of resistor that connects capacitors.

  • Cs (float) – Source capacitance value in Farads.

  • Cd (float) – Destination capacitance value in Farads.

Returns

  • rvolt (float) – Voltage across the resistor at time t.

  • vfinal (float) – Final voltage that both capacitors settle to.
