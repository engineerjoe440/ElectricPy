electricpy.seq_to_phs¶

electricpy.seq_to_phs(M012, reference='A')¶

Sequence-System to Phase-System Conversion

Converts sequence-based values to phase components.

\[M_{\text{ABC}}=A_{\text{012}}^{-1}\cdot M_{\text{012}}\]

Same as seq_to_phs.

Parameters

  • M012 (list of complex) – Sequence-based values to convert.

  • reference ({'A', 'B', 'C'}) – Single character denoting the reference, default=’A’

Returns

Mabc – Phase-based values in order of A-B-C

Return type

numpy.ndarray
