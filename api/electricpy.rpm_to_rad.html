electricpy.rpm_to_rad¶

electricpy.rpm_to_rad(rpm)¶

RPM to Radians-per-Second Converter

Given the angular velocity in RPM (Revolutions- Per-Minute), this function will evaluate the velocity in rad/sec.

Parameters

rpm (float) – The angular velocity in revolutions- per-minute (RPM)

Returns

rad – The angular velocity in radians- per-second

Return type

float
