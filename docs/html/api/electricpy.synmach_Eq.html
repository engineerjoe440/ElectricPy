synmach_Eq¶

electricpy.synmach_Eq(Vt_pu, Itmag, PF, Ra, Xd, Xq)¶

Synchronous Machine Eq Calculator

Given specified parameter set, will calculate the internal voltage on the q-axis (Eq).

\[E_q=V_{t_{pu}}-\left[R_a\cdot I_{t_{pu}}+ j\cdot X_q\cdot I_{t_{pu}}+j(X_d-X_q)\cdot I_{ad}\right]\]

where:

\[I_{t_{pu}}=I_{t_{mag}}\cdot e^{-j( \angle{V_{t_{pu}}}-\cos^{-1}(PF))}\]
\[\theta_q=\angle{V_{t_{pu}}-\left(R_a I_{t_{pu}}+j\cdot X_qI_{t_{pu}}\right)\]
\[I_{ad}=\left|I_{t_{pu}}\cdot\sin( -\cos^{-1}(PF)+\theta_q)\right|e^{j(\theta_q -90°)}\]
Parameters

  • Vt_pu (complex) – Terminal voltage in per-unit-volts

  • Itmag (float) – Terminal current magnitude in per- unit-amps

  • PF (float) – Machine Power Factor, (+)ive values denote leading power factor, (-)ive values denote lagging power factor

  • Ra (float) – AC resistance in per-unit-ohms

  • Xd (float) – D-axis reactance in per-unit-ohms

  • Xq (float) – Q-axis reactance in per-unit-ohms

Returns

Eq – Internal Synchronous Machine Voltage in per-unit-volts

Return type

complex
